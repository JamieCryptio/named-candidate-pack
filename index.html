The Work

What you'd actually be doing

Build the Advisory Intelligence toolkit

The toolkit covers the full accounting workflow — transaction-level close, chart of accounts benchmarking, policy generation, fit-gap analysis, and financial reporting outputs. Each tool combines deterministic logic with AI layers drawing on real client engagements.

Own full-stack feature delivery

From Postgres schema through API layer through React UI — no handoffs. You design the data model, build the backend pipeline, and ship the interface. All of it is yours.

Architect multi-step AI pipelines

Prompt chaining, structured output parsing, confidence scoring, failure recovery, audit-trail logging. You'll also build and extend the deterministic pre-processing layers that make AI output reliable enough for audit.

Financial document ingestion

Parse uploaded spreadsheets and structured ERP reports into typed data models for downstream AI processing. Finance practitioners will use this daily — correctness isn't optional.
